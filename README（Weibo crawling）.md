# “加班文化与职场心理健康”主题关键词微博爬虫系统
目标：爬取该主题下的关键词所有微博帖子
爬取内容：关键词、用户名、发布时间、帖子内容、转发数、评论数、点赞数、帖子链接

## 📋 功能特性

### Step 1: 输入cookie

### Step 2: 输入关键词（已预设）与设置爬取时间

### Step 3: 开始爬取

## 🚀 快速开始

### 环境要求

- Python 3.7+
- Chrome浏览器
- 相关Python依赖包

### 安装依赖

#### 基础包
pip install pandas

#### Selenium相关
pip install selenium
pip install webdriver-manager

#### 网络请求相关
pip install urllib3

#### 用于数据导出（已包含在代码中）；用于Excel导出（可选，pandas会默认安装）
pip install openpyxl  

### 基本使用

#### 代码一：按时抓取
针对数据量特别大的关键词：“加班”，“工作焦虑”，“工作疲惫”，“职业健康”，按小时抓取
代码运行后爬取该帖子当天24小时的所有帖子
例：先爬取“1月1号0时-1时”的帖子，再爬取“1月1号1时-2时”的帖子……

#### 代码二：按天抓取
针对数据量适中的关键词：“996”，“劳动法”，“加班费”，“工作压力”，“职场焦虑”，“职场压力”，“工作倦怠”，“工作过劳”，“职业倦怠”，“工作躺平”，“企业责任”，“员工关怀”，“职场内卷”，“职场竞争”，“职场PUA”，“职场霸凌”，“职场心理咨询”，“职场心理健康”，“职场关怀”
例：直接以“1月1日0时-1月2日0时”作为搜素范围，爬取1月1日所有的帖子

#### 代码三：可设置抓取时间段
此代码主要用于补充抓取前面两个代码爬取过程中出现bug的帖子
例：1月1日12时-13时的关键词“加班”帖子，在爬取过程中出现爬取超时的情况，该内容缺失；在代码中输入关键词、爬取时间，即可补充爬取这部分帖子
